How it works

Library Role Description
Lustache Presentation Renders the templates to html. Escaped/unescaped variables, sections, partials.
Fixi.js Network Updates Sends fetch requests and patches the DOM with the server's response. Dispatches events.
Hyperscript Behavior Adds client-side interactive/decorative features. Can listen and send events
css-scope-inline Styling Scopes CSS to the parent element using the me selector. Works with dynamic content.

    me {
      text-align: center;
    }

  <span id="like-count"
         _="on fx:before from #like-button
                transition my opacity to 0 over 300ms

            on fx:finally from #like-button
                transition my opacity to 1 over 400ms
                send like:updated to #like-button">
    {{likes}}
  </span>
  <br><br>
  <button id="like-button"
          fx-action="/hype/like"
          fx-method="post"
          fx-trigger="click"
          fx-target="#like-count"
          fx-swap="innerHTML"
          _="on fx:before from me toggle @disabled on me
             on like:updated toggle @disabled on me">
    Like
  </button>
</div>
