How it works

Library Role Description
Lustache Presentation renders the template to html: initial like count from the server.
Fixi.js Network Updates sends a POST request to /hype/like, then replaces the inner HTML of #like-count with the server's response without page reload.
Hyperscript Behavior handles animations: fades the counter in/out when the number changes (on mutation), and scales the button on hover (mouseenter/mouseleave).
