How this works:
Mustache - renders the initial like count from the server.
Fixi.js - sends a POST request to /hype/like without reloading, then replaces the inner HTML of #like-count with the server's response.
Hyperscript - handles animations: fades the counter in/out when the number changes (on mutation), and scales the button on hover (mouseenter/mouseleave).
