What This Demo App Uses

This demo app is built using Astra, a web server runtime for Lua. It showcases several features and libraries:

  • Middleware: Introducing patterns for middleware.
  • Routes Grouping: DSL to register a group of routes in one place.
  • Templating with Lustache: Full overview of Mustache templates in pure Lua.
  • LugSQL: Turns parameterized SQL into Lua functions.
  • Preman: Single-file HTML API client. Grug-approved..

Check out the Astra Trails GitHub repository for more details.
